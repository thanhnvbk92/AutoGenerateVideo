1 Chuẩn Bị & Cấu Hình
Hoàn thành
  • Khởi tạo cấu trúc thư mục
    Tạo các thư mục backend và frontend trong workspace d:/Dev/Project/AutoGenerateVideo.
  • Thiết lập Python Backend & Thư viện phụ thuộc
    Tạo môi trường ảo, cài đặt FastAPI, MoviePy, edge-tts, google-generativeai...
  • Thiết lập File Cấu hình
    Tạo file .env và app/config.py lưu trữ các cài đặt hệ thống và API keys.
2 Phát triển Backend (FastAPI Services)
Hoàn thành
  • Service Sinh kịch bản dài 30-40 cảnh (Gemini API)
    Viết script_gen.py để nhận prompt và generate ra kịch bản chi tiết JSON đạt 7-8 phút.
  • Service Sinh giọng nói (Edge-TTS)
    Viết tts_gen.py tải file giọng đọc tiếng Việt tự nhiên miễn phí cho từng phân cảnh.
  • Service Sinh ảnh nhân vật đồng nhất (Pollinations AI)
    Viết image_gen.py dùng Fixed Seed và Base Character Prompt để tải ảnh từ Pollinations AI.
  • Service Ghép nối & Render Video (MoviePy & FFmpeg)
    Viết video_comp.py ghép ảnh, hiệu ứng zoom, lồng tiếng, phụ đề động, trộn nhạc nền.
  • Tạo các Endpoints API & Background Tasks
    Cài đặt API điều phối toàn bộ quy trình sinh video bất đồng bộ và lấy status thời gian thực.
3 Phát triển Frontend (React + Vite)
Hoàn thành
  • Khởi tạo React + Vite & Thư viện UI
    Tạo folder frontend, cài đặt cấu hình cơ bản.
  • Hệ thống Style Premium Dark Glassmorphism
    Viết index.css với các style màu sắc, dải neon, bo góc và hiệu ứng chuyển cảnh mượt mà.
  • Giao diện Thiết lập cấu hình (StepForm)
    Form thiết lập chủ đề, giọng nói, mô tả nhân vật để làm Base Prompt và Seed.
  • Giao diện Biên tập kịch bản phân cảnh (ScriptEditor)
    Hiển thị danh sách 30-40 cảnh, hỗ trợ sửa đổi trực tiếp text của từng cảnh.
  • Giao diện Tiến trình Timeline (ProgressBar)
    Giao diện loading timeline hiển thị trực quan trạng thái sinh của từng phân cảnh.
  • Giao diện Trình phát & Tải Video (VideoPreview)
    Màn hình phát video thành phẩm, hỗ trợ tải trực tiếp về thiết bị.

Hoàn thành
  • Khởi chạy song song Backend Server (Uvicorn) và Frontend Server (Vite)

  • Chạy thử nghiệm luồng end-to-end với đề tài thực tế
    Thực hiện tạo video dài 7-8 phút, theo dõi log và kết quả render.
